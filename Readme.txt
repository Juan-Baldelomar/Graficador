

-El ejecutable cuenta con un menu para poder seleccionar si se desea graficar o correr alguno de los metodos de biseccion o newton

-El archivo fue compilado utilizando netbeans.

-Para compilar el archivo, ejecute el comando "$make -f Makefile" desde la carpeta raiz donde se encuentra el archivo Makefile. El ejecutable se generara en la carpeta "/dist/Debug/GNU-Linux/" a partir de la carpeta raiz donde se encuentra el archivo Makefile.

-para ejecutar el ejecutable "metodobiseccion" escribir en una terminal desde la carpeta donde se encuentra el archivo "$./metodobiseccion"
